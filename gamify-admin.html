🆕 Créer ma team

Crée un groupe pour réviser avec tes amis. Choisis un nom et un code secret (mot de passe), puis partage ce code à tes amis — ils n'auront qu'à le rentrer dans les paramètres de l'add-on Anki.

💡 Comment ça marche ?

1
Crée ta team ici avec un nom et un code secret de ton choix.
2
Partage le code à tes amis par message.
3
Chaque ami ouvre Anki → Outils → Gamify Settings → onglet Groupe, active le mode groupe, entre son prénom et le code.
4
Au prochain clic sur Synchroniser, le classement apparaît sur l'écran d'accueil d'Anki !
